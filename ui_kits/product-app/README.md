# Product App UI Kit

An **inferred** Kaivros product app — a wellness/VR session platform — built using the brand's tokens and component vocabulary. Since no product codebase was provided, this kit demonstrates typical brand usage across realistic screens.

## Flow
1. **Sign in** → split-layout, lilla hero panel
2. **Home** → sidebar nav, greeting, session cards
3. **Session** → full-bleed violet environment with breathing circle

## Components
- `Sidebar` — 240px left nav with streak card
- `TopBar` — date label + avatar
- `HomeView` — greeting, featured session, grid
- `SessionView` — three phases (ready / running / done)
- `SignIn` — two-column login

## Disclaimer
This UI kit is a **design extrapolation**, not a recreation. It uses the Kaivros tokens faithfully but the product itself (screens, flows, copy) is invented from the "wellness/VR platform" brief. If the real product codebase is available, replace this kit.
